# Telegram Bot (PRO++)

Бот с капчей и модерацией для групп.

## Функции:
- Капча при входе
- Блок сообщений до проверки
- Админ команды (/ban /mute)
- Антиспам
- SQLite база пользователей

## Запуск:
1. Добавить BOT_TOKEN в переменные окружения
2. Запустить main.py

## Стек:
- Python 3.11
- aiogram 3
- Railway hosting
